# Proposition solution pour l'advent of code

Mes proposition de code pour l'advent of code streamer en direct sur twitch (https://www.twitch.tv/paradev).

Pour chaque jour, il y a au moins:
- main.py qui donne la réponse de la partie 1,
- main.py qui donne la réponse de la partie 2,
- input.txt l'entrée du problème.

les finchier .bak(x).py donne des solution qui ont été abandonner.

L'enregistrement des lives est disponible sur la chaine de VOD : https://www.youtube.com/@paradoxismeVOD
